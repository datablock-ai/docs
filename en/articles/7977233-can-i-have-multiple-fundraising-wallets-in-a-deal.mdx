---
title: "Can I have multiple fundraising wallets in a deal?"
description: "Learn about fundraising wallet limitations and alternatives in Spring deals."
---

It is currently only possible to have a single fundraising wallet in a deal.

However, you can still collect funds on multiple EVM chains on the one fundraising wallet you set.

## Alternative Solution

If you want to receive funds from different wallet addresses while raising funds, the only way to enable this is by creating separate deals.
