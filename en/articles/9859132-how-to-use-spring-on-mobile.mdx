## Signing in and investing with mobile

### The best way to use Spring on mobile

To ensure the best connection, we recommend downloading the wallet extension you are using and using the browser inside this application. Every wallet extension has its own in-app browser, which makes it easy to navigate.

You can download the wallet extension you are using in the AppStore or Google Play Store.

In the demo above, the investor is using the MetaMask in-app browser.
